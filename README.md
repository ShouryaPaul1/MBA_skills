MBA Skills Toolkit (LLM-Powered)

A curated collection of practical, ready-to-use skills designed to help MBA students with academics, assignments, case studies, and interviews using modern LLMs.

## Overview

This repository contains structured prompts, workflows, and frameworks that can be applied using your preferred Large Language Model (LLM), such as ChatGPT or Claude.

It is intended for MBA students who want to:

* Improve the quality of their assignments and case work
* Prepare effectively for interviews
* Think in a more structured and analytical way
* Save time on repetitive or unstructured tasks

These skills are designed to help you produce clearer, more structured, and higher-quality outputs.

## What You Will Find

### Academic and Assignment Support

* Case study breakdown frameworks
* Market research structuring
* Business model analysis
* Report writing prompts

### Interview Preparation

* Product Manager interview frameworks
* Guesstimate solving approaches
* Behavioral question structures
* Mock interview prompts

### Strategy and Problem Solving

* First-principles thinking templates
* Problem decomposition methods
* Go-to-market strategy frameworks
* Competitive analysis structures

### Productivity Tools

* Structured note-taking prompts
* Slide and storyline creation
* Summarization workflows

## How to Use

1. Select a skill relevant to your task
2. Copy the prompt or framework
3. Use it in your preferred LLM
4. Add context specific to your problem
5. Refine the output as needed

These are starting points. The quality of output improves with better inputs and iteration.
